<h1 align="center">🌴 AuraCityBot 🌴</h1>
<p align="center">Willkommen bei deinem automatisierten Helfer für Discord!</p>

---

<h2 align="center">🚀 Funktionen</h2>

<p align="center">
  <strong>Automatisierung von Serveraufgaben</strong>: Erledigt repetitive Aufgaben wie Rollenvergabe, Moderation, Begrüßungen und mehr.<br>
  <strong>Datenbank-Integration</strong>: Arbeitet mit einer SQLite-Datenbank, um Benutzerdaten und Server-Informationen zu speichern.<br>
  <strong>Leichte Erweiterbarkeit</strong>: Das Projekt ist modular aufgebaut und lässt sich leicht um weitere Features erweitern.
</p>

<h2 align="center">🛠️ Technologien</h2>

<p align="center">
  <strong>Programmiersprache</strong>: <a href="https://www.python.org/">Python</a> – Ideal für Bots und Automatisierung.<br>
  <strong>Datenbank</strong>: <a href="https://www.sqlite.org/">SQLite</a> – Eine einfache, dateibasierte Datenbank.

<h2 align="center">⚙️ Installation</h2>

<p align="center">Folge diesen Schritten, um AuraCityBot auf deinem Server einzurichten:</p>


1. **Repository klonen**  
   ```bash
   git clone https://github.com/DerTarzan/AuraCityBot.git
   ```
   
2. **Abhängigkeiten installieren**  
   Stelle sicher, dass alle benötigten Python-Pakete installiert sind:  
   ```bash
   pip install -r requirements.txt
   ```

3. **Berechtigungen setzen**  
   Mach die Start-, Stopp- und Neustart-Skripte ausführbar:  
   ```bash
   chmod +x ./start.sh
   chmod +x ./stop.sh
   chmod +x ./restart.sh
   ```

4. **Anwendung starten**  
   Starte den Bot mit:  
   ```bash
   ./start.sh
   ```
